## Akane Framework v3

*a PHP based framework for all purpose website types.*

Sebuah Framework sederhana berbasis PHP untuk semua tipe website

Version: 3.0.2
Last Update: 2017-10-08
Author: [WebHade Creative](http://www.webhade.id)
___

### Kebutuhan Minimum

1. PHP 5.3.9 keatas
2. MySQL / MariaDB
3. Composer
